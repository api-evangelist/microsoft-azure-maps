## APIs

### Azure Maps API
Azure Maps provides a comprehensive suite of geospatial REST APIs including geocoding, reverse geocoding, route planning, traffic data, weather forecasts, elevation data, geofencing, spatial operations, and interactive map rendering. It supports multiple transport modes and real-time traffic information.

**Human URL:** [https://learn.microsoft.com/en-us/rest/api/maps/](https://learn.microsoft.com/en-us/rest/api/maps/)
